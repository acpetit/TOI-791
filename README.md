# TOI-791
Notebook reproducing the TTV analysis done for the ASTEP characterization of the two planets in the TOI-791 system.

The MCMC sampling can be reprodced in file [https://github.com/acpetit/TOI-791/blob/main/toi791_ttv_iidnormal_newtransits_rev2_public.ipynb](toi791_ttv_iidnormal_newtransits_rev2_public.ipynb) after installing in your python environment the following packages:

`pip install jnkepler, numpyro, numpy, rebound, matplotlib, arviz, xarray, seaborn, dill, corner`
